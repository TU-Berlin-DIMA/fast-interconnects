Expand description

Conversion into a CUDA iterator.

By implementing IntoCudaIterator for a type, you define how the type is converted into an iterator capable of executing CUDA functions on a GPU.

The iterator must define a transfer strategy.

The chunk_len parameter specifies the granularity of each data transfer from main-memory to device memory. The same granularity is used when passing input parameters to the GPU kernel.
